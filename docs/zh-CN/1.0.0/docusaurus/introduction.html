Introducing Docusaurus

我们很高兴推出 Docusaurus 来帮助您管理一个或多个开源网站。

由于以下原因，我们创建了 Docusaurus：

  1. 把重点放在编写好的文档，而不是担心网站的基础设施。
  2. 提供我们许多开源网站需要的功能，如博客支持，搜索和版本控制。
  3. 为了便于同时向所有人推送更新，新功能和错误修复。
  4. 最后，在所有开源项目中提供一致的外观和感觉。
