Memperkenalkan Docusaurus

Kami sangat senang memperkenalkan Docusaurus untuk membantu Anda mengelola satu atau banyak situs web sumber terbuka.

Kami membuat Docusaurus karena alasan berikut:

  1. Untuk menempatkan fokus pada penulisan dokumentasi yang baik daripada mengkhawatirkan infrastruktur situs web.
  2. Untuk menyediakan fitur yang banyak dari situs web sumber terbuka kami membutuhkan dukungan blog, pencarian dan pembuatan versi.
  3. Untuk memudahkan mendorong pembaruan, fitur baru, dan perbaikan bug untuk semua orang sekaligus.
  4. Dan, akhirnya, untuk memberikan tampilan dan nuansa yang konsisten di semua proyek sumber terbuka kami.

Ucapan terima kasih

Docusaurus tidak akan ada tanpa karya dari tim inti Docusaurus lainnya: Eric Nakagawa, Hector Ramos, Eric Vicenti dan Frank Li - mantan magang di Facebook yang menerapkan teknologi inti dan fitur.

Khusus terima kasih juga keluar untuk kami awal pengadopsi dari Docusaurus:

  • BuckleScript
  • FastText
  • Bersenda gurau
  • Jadikan Ini Terbuka
  • Lebih cantik
  • Reaksi-reaksi
  • Bereaksi Bawaan
  • Menyampaikan

Tanpa dedikasi mereka untuk membuat atau memindahkan situs web mereka ke platform, kita tidak akan berada di posisi di mana kita hari ini.
